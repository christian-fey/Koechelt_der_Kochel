Datenschutzerklärung

Deine Datenschutzerklärung sieht schon gut aus, aber ich habe einige kleinere Korrekturen und Anregungen, um sie klarer und vollständiger zu machen. Hier sind meine Vorschläge:

Welche personenbezogenen Daten werden beim Besuch unserer Website zu welchem Zweck gesammelt?

Bei der angebotenen Homepage handelt es sich um eine statische Website, deren Zweck es ist, Windprognosen zu veröffentlichen. Soweit möglich, verzichten wir auf die Erhebung personenbezogener Daten.

Allerdings werden beim Besuch unserer Website von unserem Host GitHub standardmäßig Informationen wie IP-Adresse, Standort, Zugriffsdatum und Browser-Typ in sogenannten Log-Daten protokolliert und gespeichert. Diese Daten werden erhoben, um potenzielle Angriffe abzuwehren und zur Erstellung statistischer Analysen, um die Website zu verbessern. Wir selbst haben keinen Zugriff auf die personenbezogenen Daten. Für weitere Informationen siehe GitHub Pages und die GitHub Datenschutzrichtlinie.

Eingebettete Inhalte

Um die Zahl der Aufrufe unserer Website zu messen, haben wir einen Counter über die externe URL https://hits.seeyoufarm.com/ eingebunden. Laut Entwickler werden hierbei jedoch keine personenbezogenen Daten erhoben ("HITS counts every page hit without storing sensitive information (IP, headers, etc.)"), siehe Hit Counter auf GitHub.

Cookies

Wir haben der Website keine Cookie-Funktionen hinzugefügt, und unseres Wissens nach werden solche auch nicht durch unseren Host GitHub gesetzt.
